***** 파이썬 리스트 중복 제거 *****
b = [1, 2, 3, 3, 4, 4, 5]
b = list(set(b))
b => [1, 2, 3, 4, 5]
set은 중복을 허용하지 않기 때문에 set로 바꿔준뒤 list로 바꾸면 중복제거가능.

***** 파이썬 얕은복사 깊은복사 *****
https://wikidocs.net/16038 참조
a = [1, 2]
b = a 일때, id값이 서로 같다.
따라서 a의 값을 변경하면 b의 값도 변경된다.
위 문제를 해결하기 위해서는 copy메소드를 이용해야 한다.
import copy
b = copy.deepcopy(a)

***** 리스트 컴프리헨션 *****
예시 1 - 리스트 생성
[x for x in range(10)]
range(10)을 중간 x에 넣고, 그 x를 출력함
예시 2 - 받은 값 출력
[print(int(i)-a*b)for i in input().split()]
input한 값들을 스플릿해서 하나씩 i에 넣고, 문자열일테니 int()로 바꿔주면서 a*b을 뺀 값을 print함
예시 3 - 조건 걸기
[x for x in range(1, 10+1) if x % 2 == 0]
-> [2, 4, 6, 8, 10]
예시 4 - 중복 for
[(x, y) for x in ['a', 'b', 'c'] for y in ['a', 'b', 'c']
-> ['a', 'a'], ['a', 'b'], ['a', 'c'], ['b', 'a']...
예시 5 - 중복 if
[x for x in range(10) if x < 5 if x % 2 == 0]
-> [0, 2, 4]
예시 6 - 입력 수만큼 입력받기
a = [list(map(int,input().split())) for _ in range(n)]

***** 한 칸 띄워 출력하기 *****
a = [1, 2, 3]
b = [2 ,3, 4]
리스트 값만 출력하고 싶다면
[print(i, end = ' ') for i in a]
[print(i, end = ' ') for i in b]
를 쓰면 된다. 하지만 출력은
1 2 3 2 3 4가 나오고, \n을 쓰면
1 2 3

2 3 4
두칸이 띄워진다. 방법은 사이에
print('')를 넣으면
1 2 3
2 3 4
정상적으로 출력됨.
